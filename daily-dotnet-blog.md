## 🚀 GitHub Copilot SDK Now Available in Public Preview

**Major AI Development News**

GitHub has released the Copilot SDK in public preview, giving developers the ability to embed Copilot's agentic capabilities directly into their own applications. The SDK is now available in five languages including .NET (`dotnet add package GitHub.Copilot.SDK`).

Key features include:
- Custom tools and agents with domain-specific tool definitions
- Fine-grained system prompt customization
- Streaming and real-time responses
- Blob attachments for images and binary data
- OpenTelemetry support with W3C trace context propagation
- **Bring Your Own Key (BYOK)** for Azure AI Foundry, OpenAI, and Anthropic

This is a significant shift for .NET developers looking to build AI-powered applications without reinventing the wheel.
